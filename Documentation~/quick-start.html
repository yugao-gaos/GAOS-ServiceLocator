Quick Start Guide

Get started with GAOS Service Locator in minutes

Installation

Add the package to your Unity project via the Package Manager:

{
    "dependencies": {
        "com.gaos.servicelocator": "1.2.2"
    }
}

Basic Usage

1. Define Your Service Interface

Start by defining an interface for your service. This interface will define the contract that your service implementation must follow:

public interface IGameSettingsService
{
    float MasterVolume { get; set; }
    bool IsFullscreen { get; set; }
}

2. Implement the Service

Create a class that implements your service interface. Use the Service attribute to mark it as a service:

[Service(typeof(IGameSettingsService), "GameSettings", ServiceLifetime.Singleton, ServiceContext.Runtime)]
public class GameSettingsService : IGameSettingsService
{
    public float MasterVolume { get; set; }
    public bool IsFullscreen { get; set; }
}

About the Service Attribute

  • typeof(IGameSettingsService): The interface this service implements
  • "GameSettings": A unique name to identify this implementation
  • ServiceLifetime.Singleton: (Optional) Determines the service lifetime:
    • Singleton: One instance shared across the application
    • SceneSingleton: One instance per scene, disposed when scene unloads
    • Transient: New instance created each time
    • SceneTransient: New instance created each time, all disposed when scene unloads
  • ServiceContext.Runtime: (Optional) Specifies when the service is available

3. Use the Service

Retrieve and use the service anywhere in your code:

var settings = ServiceLocator.GetService<IGameSettingsService>("GameSettings");
settings.MasterVolume = 0.8f;
settings.IsFullscreen = true;
