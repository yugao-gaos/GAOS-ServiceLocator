A modern, Unity-optimized service management solution. Read the development story to learn about the journey and philosophy behind this package.

Introduction

GAOS Service Locator provides a powerful yet lightweight dependency management solution built specifically for Unity. It seamlessly integrates with Unity's component lifecycle while delivering thread-safe performance, flexible service configurations, and intuitive dependency resolutionâ€”all without the configuration complexity of traditional dependency injection frameworks.

Unity-Optimized Architecture

  • Seamless MonoBehaviour and ScriptableObject integration
  • Scene-aware service management with automatic cleanup
  • Support for both runtime and editor-only services
  • Smart handling of scene transitions

High Performance

  • Thread-safe operations for multi-threaded systems
  • Service pooling for object reuse
  • Async service support with Task-based API
  • Efficient memory management

Developer Experience

  • Attribute-based registration with minimal boilerplate
  • Flexible service lifetimes (Singleton, Transient, Scene-based)
  • Comprehensive validation and debugging tools
  • Clear API with helpful diagnostics
